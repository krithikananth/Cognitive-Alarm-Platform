# Cognitive Alarm Platform

## Project Description
The Cognitive Alarm Platform is a web application that helps users manage alarms and reminders efficiently.

## Features
- User Authentication
- Alarm Management
- Dashboard
- Responsive UI
- Docker Support

## Technologies Used
- React.js
- Python (Flask)
- Docker
- Git & GitHub

## Project Structure
```
backend/
frontend/
docs/
README.md
docker-compose.yml
```

## How to Run

1. Clone the repository
2. Run:
```
docker-compose up --build
```
3. Open:
```
http://localhost:3000
```
